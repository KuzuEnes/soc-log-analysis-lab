# SOC Log Analysis Lab

Bu repo, SOC Analyst / Blue Team alanında gelişmek için oluşturduğum kişisel öğrenme ve portföy reposudur.

## Amaç

Bu çalışmada saldırıların loglarda nasıl göründüğünü, alert triage sürecini, false positive / true positive ayrımını ve temel detection mantığını öğreniyorum.

## Öğrenilecek Konular

- SOC analyst temel kavramları
- Log analizi
- Windows Event Logs
- Linux auth logs
- Web access logs
- Brute force tespiti
- Web scan tespiti
- MITRE ATT&CK mapping
- Python ile log analizi
- Detection rule yazımı
